# Heterogeneous Credential Verification for Workload and Agentic Systems

This is the working area for the individual Internet-Draft, "Heterogeneous Credential Verification for Workload and Agentic Systems".

* [Editor's Copies](https://agent-security-labs.github.io/wimse-heterogeneous-credential-draft/)
* [Datatracker Page](https://datatracker.ietf.org/doc/draft-jiang-wimse-heterogeneous-credential)
* [Individual Draft](https://datatracker.ietf.org/doc/html/draft-jiang-wimse-heterogeneous-credential)
* [Compare Editor's Copy to Individual Draft](https://agent-security-labs.github.io/wimse-heterogeneous-credential-draft/#go.draft-jiang-wimse-heterogeneous-credential.diff)


## Demo

The [`demo/`](demo/) directory holds a proof-of-concept demonstration of the
heterogeneous credential verification flow described in this draft:

* `heterogeneous-credential-demo.pptx` — slides that walk through the demo.
* `heterogeneous-credential-demo-linux` / `heterogeneous-credential-demo-macos` /
  `heterogeneous-credential-demo-windows.exe` — standalone demo executables for
  Linux, macOS, and Windows.

These are illustrative artifacts and are not part of the Internet-Draft itself.

## Command Line Usage

Formatted text and HTML versions of the draft can be built using `make`.

```sh
$ make
```

Command line usage requires that you have the necessary software installed.  See
[the instructions](https://github.com/martinthomson/i-d-template/blob/main/doc/SETUP.md).
